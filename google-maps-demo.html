üîó Available API Endpoints

POST /api/maps/geocode

Convert address to coordinates

Request:
{
  "address": "Brandenburger Tor, Berlin"
}
POST /api/maps/reverse-geocode

Convert coordinates to address

Request:
{
  "lat": 52.5200,
  "lng": 13.4050
}
POST /api/maps/nearby-places

Find nearby restaurants, schools, etc.

Request:
{
  "lat": 52.5200,
  "lng": 13.4050,
  "type": "restaurant",
  "radius": 1000
}
POST /api/maps/search-by-location

Search properties near a location

Request:
{
  "location": "Alexanderplatz, Berlin",
  "radius": 5
}
POST /api/maps/distance

Calculate distance between two points

Request:
{
  "origin": "Berlin, Germany",
  "destination": "Munich, Germany",
  "mode": "driving"
}
GET /api/maps/place-types

Get available place types for search

No request body required
